# IndiaAI MSME Intelligent Matching System

AI-powered multimodal onboarding and MSME-to-SNP matching platform built for the IndiaAI Innovation Challenge.

## Features
- Voice-based MSME onboarding (multilingual)
- Automatic product attribute extraction
- ONDC category classification
- Intelligent SNP matching using semantic search + learning-to-rank
- Explainable AI rationale
- DPDP-compliant consent and data handling
- Self-improving feedback loop

## Tech Stack
- FastAPI backend
- Next.js frontend
- Sentence Transformers + LightGBM ranking
- FAISS vector search
- VOSK multilingual ASR
